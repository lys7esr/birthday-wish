## 🚀 How It Works

The website starts with a **hacker-style terminal animation**, builds suspense, and then transitions into a **birthday surprise screen** with photos, messages, and celebration effects. There is a music feature too:)

---

## 🎨 Customization Guide

### ✏️ Change Friend's Name

Edit in `script.js`:

```js
friendName: '[FRIEND_NAME]'
```

---

### 🖼️ Add Your Own Photos

Replace images inside:

```
/images/
```

Make sure file names match those in `index.html`.

---

### 💬 Edit Messages

* Terminal messages → `script.js`
* Birthday text → `index.html`

---

## 📱 Make It a Real Surprise (QR Idea)

1. Deploy using GitHub Pages or Netlify
2. Copy your website link
3. Generate a QR code
4. Share or print it

👉 When they scan it = surprise 🎉
---

## 📲 Optional: WhatsApp Integration

For privacy reasons, WhatsApp functionality is disabled by default.

To enable:

1. Uncomment in `script.js`
2. Add your number and message
3. Enable the button in `index.html`

⚠️ Use only permitted contact details.
